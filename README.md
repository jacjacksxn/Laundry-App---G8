# Laundry App - G8

Android App Development Groupwork for Group 8.

## Overview

Laundry App is an Android application designed to help users manage laundry-related tasks in a simple and organized way. This project was built as part of an Android app development group assignment.

## Features

- User-friendly Android interface
- Laundry task management
- Group project structure
- Built with Java

## Tech Stack

- **Language:** Java
- **Platform:** Android
- **Project Type:** Groupwork / Academic project

## Getting Started

### Prerequisites

- Android Studio
- Java Development Kit (JDK)
- Android SDK

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jacjacksxn/Laundry-App---G8.git
   ```
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Build and run the app on an emulator or Android device.
